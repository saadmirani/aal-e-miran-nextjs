## 🚀 Ready to Use

### Quick Start (3 commands)
```bash
cd aal-e-miran-nextjs
npm install
npm run dev
```

Then visit:
- Home: `http://localhost:3000`
- Biography: `http://localhost:3000/biography/rahman-bakhsh-qadri`
- Book: `http://localhost:3000/books/kitab-ul-ansab`

## 📝 Adding Content (Very Easy!)

### Add New Biography
1. Create: `content/biographies/your-slug/`
2. Create: `en.md` and `ur.md`
3. Available at: `/biography/your-slug`

### Add New Book
Same process in `content/books/`

## 🌍 Deployment to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Add env: `NEXT_PUBLIC_DOMAIN=bazmesaadaat.com`
4. Deploy!

Automatic updates on every GitHub push.

## 🔄 Domain Switching

Change `NEXT_PUBLIC_DOMAIN` env variable in Vercel = instant domain change everywhere!
